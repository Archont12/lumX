Icon

lx-icon is a directive that will become an icon.

LumX provide Material Design iconic font based on Austin Andrews work. As lx-button, you can specify size, color and type variants.
Parameter Type Default Description
lx-color string Define the button color. Options: colors defined in colors and sizes section in dist/scss/core/settings/defaults.scss
lx-id* string The icon identifier tha tcan be found in Material Design Icons website.
lx-size string Define the button size. Options: xs, s, m, l, xl
lx-type string Define the button type. Options: raised, flat, fab, icon
