Tabs

lx-tabs is a directive that create a full tabs navigation.

You will need to transclude one lx-tab directive per tab with tab label or icon as parameter and pane content as transcluded elemnt. Moreover, you can define the active tab an disable each tab. You can specify the tabs color with main colors defined in dist/scss/core/settings/defaults.scss.
If theme is light, color will be applied to tabs label / icon and indicator. If theme is dark, color will be applied to tabs background and tabs label / icon will be white.
Moreover, you can use a label or an icon for each tab. You can use ng-repeat to loop threw an array of tabs. Then, you will be able to manage tabs easily. You can separate tabs and pane with two directives: lx-tabs for tabs and lx-tabs-panes for panes.
It's usefull when you need to scroll into panes with fixed tabs.
Parameter Type Default Description
lx-layout string full Specify the tabs layout. Options: full (tabs will share the available width), inline (tabs will be aligned to the left)
lx-theme string light Specify the tabs theme. Options: light (color will be applied to tabs label and indicator), dark (color will be applied to tabs background and tabs label / icon will be white)
lx-color string primary Define the tabs color according to the theme. Options: colors defined in colors and sizes section in dist/scss/core/settings/defaults.scss
lx-indicator string accent Define the indicator color when theme is set to dark. Options: colors defined in colors and sizes section in dist/scss/core/settings/defaults.scss
lx-active-tab integer Define the active tab index.
lx-panes-id string Target the right lx-tabs-panes id in seaparate mode.
links object Define the tabs links in seaparate mode.
